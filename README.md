# 月老.skill

「你连自己要什么都说不清楚，月老的红线往哪牵？」

相亲前问你"你想找什么样的"，你说"随缘"。
见完三个发现——一个太吵，一个太冷，一个聊不到一块。
不是缘分没到，是你根本没想清楚。

月老不帮你算命。月老帮你画画像。
七轮对话，从红线到生活方式，把你脑子里那团模糊的"感觉"拆成一份系统化的择偶标准。
不是列清单，是帮你搞清楚自己到底在意什么。

看效果 · 安装 · 七轮对话 · 诚实边界

## 效果示例

没有月老的时候：

> 你想找什么样的？
>
> 嗯……性格好吧，温柔一点，然后有共同语言……差不多就行。

——"差不多就行"是你没想清楚，不是你要求低。

有了月老之后，经过七轮对话，输出一份完整画像：

```markdown
# 择偶画像

## 用户基本信息
女，1998年3月，165cm，华东师范大学本科，上海互联网产品经理，年入35-40万，计划在上海长期发展

## 红线
- 男，95-01年（±3岁），170cm以上，BMI正常
- 本科及以上学历
- 上海发展或愿意来上海，不接受长期异地
- 不抽烟，不酗酒，无不良嗜好
- 父母关系和睦，家庭结构简单

## 核心性格期望
- **[必须有]** 情绪稳定，能理性沟通
- **[必须有]** 有责任感，说到做到
- **[很重要]** 有上进心，对未来有规划
- **[很重要]** 尊重女性，不大男子主义
- [加分项] 幽默感，能让生活有趣

## 相处模式
- **冲突处理**：冷静10-20分钟后沟通，不冷战过夜
- **日常相处**：工作日各自独立，周末一起活动
- **家庭财务**：建立共同账户+各自私房钱，大额支出共同商量
- **生育规划**：希望有1-2个孩子，30-32岁考虑生育
- **家庭角色**：双方事业并重，家务共同分担

## 生活方式
- 周末喜欢出门探店、看展、短途旅行，不是纯宅
- 口味偏清淡，希望对方会做饭或愿意学
- 有运动习惯，每周至少运动2次
- 早睡型，11点前睡觉
- 喜欢猫，接受养宠物

## 情绪触发点
- **正面期望**：低谷期愿意陪伴倾听，不急着给建议；记得我说过的小事
- **负面雷区**：冷暴力、翻旧账、拿我和别人比较、承诺了做不到

## 兴趣爱好
- 我的兴趣：阅读、瑜伽、咖啡探店、旅行、摄影
- 希望有交集：旅行、探店、阅读（至少能聊）
- 可接受差异：游戏、运动类型（不强求一致）

---
生成日期：2026-04-27
```

拿着这份画像去相亲，你至少知道什么该直接pass，什么可以再聊聊。

## 七轮对话

月老不是一上来就让你列条件。先聊你，再聊你想找谁。

| 轮次 | 名称    | 问什么                   | 为什么先问这个             |
| -- | ----- | --------------------- | ------------------- |
| 1  | 基本信息  | 你的性别、年龄、身高、学历、城市、收入   | 画像是照着你的镜子画的，得先知道你是谁 |
| 2  | 红线条件  | 不可妥协的硬性标准             | 先画边界，才知道选择空间有多大     |
| 3  | 性格期望  | 你最看重的性格特质，排个序         | 性格决定日后的80%矛盾源       |
| 4  | 相处模式  | 吵架怎么处理？家务怎么分？过年去谁家？   | 恋爱靠心动，相处靠模式         |
| 5  | 生活方式  | 宅家还是出门？做饭还是外卖？早睡还是熬夜？ | 日常不合比三观不合更磨人        |
| 6  | 情绪触发点 | 什么行为让你安心？什么行为让你崩溃？    | 雷区比喜好更值得提前知道        |
| 7  | 兴趣爱好  | 你喜欢什么？希望有交集吗？         | 兴趣不需要完全一样，但得有对话的入口  |

整个流程约20-35分钟。每个阶段都有引导问题帮你思考，不是让你对着空白文档发呆。

## 月老做了什么

其他择偶工具给你一张表格，让你填空。你填着填着就变成了列购物清单。

月老做的是**渐进式追问**——

| 别人         | 月老                   |
| ---------- | -------------------- |
| "你的要求是什么？" | "你最不能接受什么？"          |
| "列出所有条件"   | "如果只能保留三条红线，你留哪三条？"  |
| "排序1-10"   | "想象一个有A但没B的人，你能接受吗？" |
| 填完就完了      | 填完检查一遍：这些条件有矛盾吗？现实吗？ |

月老帮你区分三件事：

- **红线**——没有，直接pass，没有商量余地
- **核心期望**——权重最高的2-3项，决定长期幸福
- **加分项**——有了更好，没有也行

很多人相亲低效，不是因为标准太高，是因为分不清这三类——把加分项当红线，把红线当"看感觉"。

## 诚实边界

月老能做的：

- 通过结构化对话帮你梳理择偶标准
- 帮你区分"必须有"和"有了更好"
- 在条件之间发现矛盾并提醒
- 当条件过于理想化时温和提醒现实可行性
- 输出一份可以直接用于筛选的结构化画像

做不到的：

| 维度     | 说明                    |
| ------ | --------------------- |
| 预测缘分   | 画像帮你缩小范围，但不会告诉你"就是ta" |
| 替你做决定  | 标准是你定的，月老只做记录和引导      |
| 保证匹配成功 | 画像是筛选工具，不是包婚书         |
| 覆盖所有维度 | 七轮对话覆盖核心，但感情的化学反应没法量化 |

一个不告诉你局限的择偶工具，和催你结婚的亲戚没区别。

## 设计原则

1. **引导而非主导** — 给选项启发思考，决定权始终在你
2. **逐步深入** — 从硬性条件到软性期望，从理性到感性
3. **区分优先级** — 帮你识别"必须有"、"很重要"和"加分项"
4. **现实检验** — 条件太理想化时温和提醒，不评判但不说谎
5. **尊重选择** — 你的标准就是你的标准，月老只记录不审判
6. **可执行输出** — 画像足够具体，带去相亲能直接用

## 使用画像的正确姿势

生成画像后——

1. **定期回顾** — 每3-6个月看一次，你在成长，标准也会变
2. **框架不是枷锁** — 红线是硬的，其他都可以聊
3. **抓大放小** — 重点关注红线和核心性格，生活方式可以磨合
4. **带画像去相亲** — 不是带检查表打分，是心里有底知道什么该pass
